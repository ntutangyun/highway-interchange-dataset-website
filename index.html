Perception Error Modeling

A Perception Error Modeling Framework for Autonomous Driving System Testing

Perception Error Demos

This section presents the video demonstrations of the basic errors applied to a single obstacle scenario.

  • For each test case, its corresponding scenario configuration (including the perception error information) is presented on the right side in JSON format.
  • The perception error information is at "SCENARIO" -> "ego_vehicle" -> "perception_errors".
  • The actual perception data sent to the "Plan" part of Apollo is highlighted in light green, while the ground truth perception is as shown in dark red, which is only for visualization purpose and invisible to the "Plan" part.

Hardware: Detection Range Error

The simulates the scenario where the detection range changes during the test case. Obstacles are filtered out if they are out-of-range.

Hardware: Sensor Alignment Error

The simulates the scenario where the sensor (say LiDAR) becomes misaligned. It by default affects all the obstacles' position and orientation.

Software: Position x Uniform Random Error

The simulates the scenario where there is a random error added to the position's x-axis the selected obstacle.

Software: Position y Gaussian Error

The simulates the scenario where there is a gaussian error added to the position's y-axis the selected obstacle.

Software: Theta Uniform Random Error

The simulates the scenario where there is a random error added to the obstacle's rotation.

Software: Dimension Length Uniform Random Error

The simulates the scenario where there is a random error added to the obstacle's length dimension.

Software: Dimension Width Uniform Random Error

The simulates the scenario where there is a random error added to the obstacle's width dimension.

Software: Velocity Uniform Random Error

The simulates the scenario where there is a random error added to the obstacle's velocity (speed).

Software: ID Math Operation Error

The simulates the scenario where the ID of the obstacle is changed due to tracking inconsistency.
